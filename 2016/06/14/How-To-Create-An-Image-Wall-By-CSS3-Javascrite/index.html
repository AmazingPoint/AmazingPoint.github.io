How To Create An Image Wall By CSS3&Javascrit

Posted on 2016-06-14   |  

For a long time, That I think JavaSacript is hard to use it to write a pretty HTML page. The reason of this is That HTML and CSS always display diffrence with diffrent browser.

But Now, I’ll say bye bye to IE6(7-8), I use HTML5 and CSS3!

This day, I’ll write a page for my best friend! She has lots of beautiful picture which is taked when she arrive a new pleace! I’ll show these pictures with a html page!

Have a glance at demo

What do you see?

You can see a webpage strewn with some pictures. Each picture is sloping， black-and-white, And have four borders which is corlored with white.

When you move mouse on a picture, This picture will rotate to level, then the picture which is been selected will have scale and shadow effect . When your mouse leave this picture, It will recover.

How To Do It With CSS3

To do this effect, you can use css3 and no javascript!
so, We look at the list below:

  • position: absolute;

  • top: 20px; left:40px;

  • transform: rotate(45deg) scale(1.5);

  • translate: 1s;

  • border: 10px solid #fff;

  • box-shadow: 10px 10px 10px #333;

  • img:hover { styles };

We can use transform:rotate(12deg) to rotate element to 12 degress, and transfrom:scale(1.2) to zoom element to 1.2 times, and border:10px 10px 15px #fff to set border width and color, box-shadow:10px 10px 10px #333 to set shadow is on top or bottom, left or right, and shadow’s width and color.

the img:hover{} to set the styles when mouse move on image.

the tranlate:1s is mean of that when mouse move on picture, the effect will coming with 1s.

So, we can use css3 to instance these effect without javascript.

TIPS margin(padding, border) use diffrent number of values.
margin(100px) // top bottom left right all are 100px
margin(100px 200px 300px) //top is 100px, left and right are 200px, bottom is 300px
margin(100px 200px 300px 400px) // top 100px, right 200px, bottom 300px, left 400px
